## 定时任务

### Heartbeat（10分钟）
- 只在开发状态时执行监控
- 日常状态（idle, completed）跳过
- 生成报告并发送给K哥

### 触发条件
```json
// 只在以下状态执行
if (config.status in ["in_progress", "reviewing"]) {
  execute_heartbeat();
}
```

## 异常处理

### 1. Agent离线

### 2. Issue超时

- 动作：接收墨汁儿通知 → 通知K哥

### 3. Git冲突
- 判定：煎饼/墨汁儿报告冲突
